## Build Setup

``` bash
# Зависимости
python >= 3.5
node.js >= 5.10
yarn >= 1.0


# Установка 

$ virtualenv -p python3 venv
$ source venv/bin/activate
$ pip install -r requirments.txt

$ yarn install


# run python backend for development
yarn run server

# serve with hot reload at localhost:8080
yarn run hot

# build for production with minification
yarn run build
```
